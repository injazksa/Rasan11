# 🛡️👑 بوابة المدير الأسطوري - الدخول السري

## السر الإمبراطوري: كيفية الوصول إلى لوحة التحكم

### الطريقة الوحيدة للدخول:

1. **توجه إلى صفحة الدخول** (`/login`)
2. **اضغط على الشعار الذهبي 7 مرات متتالية** بسرعة
3. **ستظهر بوابة المدير السرية** مع تأثير ذهبي براق
4. **أدخل بيانات حسابك الإداري** (البريد الإلكتروني وكلمة المرور)
5. **ستنقلك مباشرة إلى لوحة التحكم المطلقة** (`/rasan-master-control`)

---

## ⚙️ ميزات لوحة التحكم (God Eye Dashboard)

### 1. **عرض جميع المستخدمين والاتحادات**
- قائمة شاملة بجميع المستخدمين المسجلين في النظام
- عرض الدولة والاتحاد لكل مستخدم
- البحث والتصفية المتقدمة

### 2. **إدارة حسابات المستخدمين**
- **تغيير كلمة المرور:** اضغط على أيقونة المفتاح لتغيير كلمة مرور أي مستخدم
- **تفعيل/تعطيل الحسابات:** اضغط على أيقونة المستخدم لحظر أو تفعيل الحساب
- **عرض معلومات التواصل:** البريد الإلكتروني ورقم الهاتف

### 3. **الإحصائيات والتقارير**
- إجمالي عدد المستخدمين
- إجمالي الإيرادات
- آخر ظهور للمستخدمين

### 4. **الأمان والتحكم**
- فقط حسابات "admin" يمكنها الوصول إلى هذه البوابة
- جميع العمليات محمية بـ JWT Token
- تنبيهات أمان عند كل تعديل

---

## 🔐 معلومات أمنية مهمة

### نقاط الدخول:
- **الدخول العادي:** `/login` (للمستخدمين العاديين)
- **الدخول السري:** يتم تفعيله عند الضغط 7 مرات على الشعار في `/login`
- **لوحة التحكم:** `/rasan-master-control` (متاحة فقط لـ admin بعد التحقق)

### التحقق من الصلاحيات:
```javascript
// يتم التحقق من الصلاحيات في الخلفية (Backend)
if (data.user.role !== 'admin') {
  throw new Error('صلاحيات غير كافية. هذا الدخول مخصص للمديرين فقط.');
}
```

## 🎯 حالات الاستخدام الشائعة

### إعادة تعيين كلمة مرور مستخدم:
1. ابحث عن المستخدم في لوحة التحكم
2. اضغط على أيقونة المفتاح (Key icon)
3. أدخل كلمة المرور الجديدة
4. سيتم التحديث فوراً

### تفعيل/تعطيل حساب:
1. ابحث عن المستخدم
2. اضغط على أيقونة المستخدم (User icon)
3. سيتم تبديل الحالة بين نشط وموقوف

### البحث عن مستخدم معين:
1. استخدم شريط البحث في الأعلى
2. ابحث بالاسم أو البريد الإلكتروني أو رقم الـ ID

## ⚠️ تنبيهات أمنية

- **لا تشارك بيانات حسابك الإداري مع أحد**
- **السر (الضغط 7 مرات) معروف فقط لك**
- **جميع العمليات تُسجل في قاعدة البيانات**
- **تغيير كلمات المرور يجب أن يتم بحذر شديد**
